# Papaya
Escencialmente un wrapper alrededor de la API de modrith...
Podés ver mods, resource packs, shaders y modpacks según la relevancia.

**Funcionamiento**: Se conecta a la API de modrinth y solicita proyectos que sean o mods, resource packs, shaders, o modpacks. Luego los muestra en pantalla con scroll.

**Herramientas**: Astro, y uso TypeScript por los errores nada más.

**Instrucciones**: Correr `npm run dev` en el root del proyecto y conectarse a `localhost:4321`. [También está hosteado en vercel.](https://papaya-eight.vercel.app/)
